Observation
===========

We can create an observation:

>>> from vct.core.observation import Observation
>>> observation = Observation()
>>> observation.title = 'something'
>>> observation.title
'something'

We can store an observation

either like this:

>>> from vct.core.db.interfaces import IDatabase
>>> from zope.component import getUtility
>>> IDatabase(observation).put('frsecu', '1234')
>>> number, results = getUtility(IDatabase).get(uid=('frsecu', '1234'))
>>> results[0].uids
{'frsecu': '1234'}
>>> getUtility(IDatabase).delete('frsecu', '1234')



which would allow to have a different database for observation or other objects
